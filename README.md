# 🏎️ F1 2025 Canadian GP: Russell's Tyre Management Analysis

## 📌 Project Overview
이 프로젝트는 2025년 캐나다 그랑프리에서 조지 러셀(George Russell)이 보여준 압도적인 타이어 관리와 주행 전략을 F1 텔레메트리 데이터를 바탕으로 정량적으로 분석합니다. 막스 베르스타펜 등 경쟁자들과의 페이스 비교를 통해 러셀의 안정적인 우승 공식을 도출합니다.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** FastF1, Pandas, Matplotlib, Seaborn
- **Data Source:** Ergast API / FastF1 Timing & Telemetry Data

## 📊 Key Analysis Points
1. **Lap Time & Pace Consistency:** 스틴트별 랩타임 추이 및 타이어 마모에 따른 페이스 저하(Degradation) 비교
2. **Telemetry Comparison:** 코너 진입/탈출 시의 속도(Speed), 스로틀(Throttle), 브레이크(Brake) 데이터를 통한 드라이빙 스타일 분석
3. **Tyre Strategy:** 경쟁 드라이버들과의 피트 스톱 타이밍 및 타이어 컴파운드 효율성 분석
